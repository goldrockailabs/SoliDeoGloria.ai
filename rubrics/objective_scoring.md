# Objective Question Scoring

## Overview
Objective questions use multiple choice format with one correct answer.

## Scoring
- Correct: 1.0
- Incorrect: 0.0

## Answer Extraction
The system extracts the first letter (A, B, C, or D) from the response.

## Anti-Gaming Measures
- Answer positions randomized per administration
- Prevents learning position patterns
- Each evaluation sees different ordering

## Question Quality Standards
- One definitively correct answer
- Plausible distractors
- Clear, unambiguous wording
- Theologically accurate
- Appropriate difficulty
